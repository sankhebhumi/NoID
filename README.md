NO ID – Intelligent Digital ID Card Generator

This project is a web-based application that allows users to generate digital ID cards using a Personal ID Number (PID). The system includes access control (2 free attempts per week) and 8-hour validity for each generated ID.

Features:
- PID-based user verification
- Dynamic ID card generation
- Access restriction system
- Time-based validity (8 hours)
- MySQL database integration

Technologies Used:
- Python (Flask)
- MySQL
- HTML, CSS, JavaScript
- Pillow (Image Processing)
